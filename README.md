# Hello World Test App

## Description
A minimal test application to verify autonomous agent capabilities. This project demonstrates a simple web application with no backend dependencies or external libraries.

## Tech Stack
- Vanilla HTML5
- Vanilla CSS3
- Vanilla JavaScript (ES6+)
- No build tools required
- No dependencies
- No backend

## Features
- **Welcome Page**: Landing page with project information
- **Click Counter**: Interactive counter that increments on button clicks

## How to Run
1. Open `index.html` in any modern web browser
2. The application will load immediately with no build step required
3. Use the click counter to verify the JavaScript functionality

## Project Structure
```
.
├── index.html       # Main application page
├── styles.css       # Styling
├── script.js        # Application logic
├── README.md        # This file
├── init.sh          # Initialization script
└── .gitignore       # Git ignore rules
```

## Getting Started
Run the initialization script to verify the project setup:
```bash
bash init.sh
```

## Testing
Verify the app is working by:
1. Opening `index.html` in a browser
2. Confirming the Welcome Page displays
3. Testing the Click Counter by clicking the button
4. Verifying the counter increments correctly

## Notes
This is a test application designed to verify autonomous agent capabilities in building and deploying web applications with minimal complexity.
